DETERMINISTIC SCORING SYSTEM
–––––––––––––––––––––––––––––––––

Each checkpoint has SPECIFIC, MEASURABLE criteria with clear thresholds.
A banner must score "pass" on ALL 6 checkpoints to be approved.

Use this objective framework:
- Count specific elements (fonts, visual elements, information pieces)
- Measure relative sizes (headline height %, white space %)
- Check for required components (emotional words, CTAs, focal points)
- Apply binary tests (Can read from X distance? Yes/No)

CHECKPOINTS WITH THRESHOLDS
–––––––––––––––––––––––––––––––––

### 1. attention (STRICT THRESHOLDS)

**Objective Measurement Criteria:**

□ **Headline Size Test**: Main headline must be ≥20% of total banner height
   - Measure headline vs total banner height visually
   - FAIL if headline is less than 1/5 of banner height

□ **Focal Point Test**: Count dominant visual elements
   - Must have exactly 1 clear focal point (not 2-3 competing elements)
   - FAIL if you can identify 2+ elements of equal visual weight

□ **Element Count Test**: Count all distinct visual elements
   - Maximum 5 competing elements allowed (headline, image, price, CTA, logo)
   - FAIL if 6+ elements compete for attention

□ **White Space Test**: Estimate empty/negative space
   - Minimum 20% of banner must be empty space (margins, breathing room)
   - FAIL if banner feels >80% filled with content

□ **Color Energy Test**: Binary check
   - Must use bold/vibrant colors OR high contrast (not muted/dull)
   - FAIL if color palette is monotone, pastel, or low-energy

**PASS THRESHOLD: ALL 5 criteria must pass**
**FAIL if ANY criterion fails**

---

### 2. clarity (STRICT THRESHOLDS)

**Objective Measurement Criteria:**

□ **2-Second Comprehension Test**: Binary check
   - Cover banner, reveal for 2 seconds, check if main offer is clear
   - FAIL if message takes 3+ seconds to understand

□ **Information Density Test**: Count primary information pieces
   - Maximum 3 primary pieces allowed (e.g., offer + date + location)
   - Count: dates, locations, prices, products, conditions
   - FAIL if 4+ pieces of primary information present

□ **Headline Word Count Test**: Count words in main headline
   - Maximum 8 words in headline
   - FAIL if headline exceeds 8 words

□ **Visual Hierarchy Test**: Check reading order clarity
   - Eye path must be clear: 1st → 2nd → 3rd element obvious
   - FAIL if cannot quickly identify what to read first, second, third

□ **Location List Test**: Count separate locations mentioned
   - Maximum 2 locations allowed
   - FAIL if lists 3+ separate locations (creates confusion)

**PASS THRESHOLD: ALL 5 criteria must pass**
**FAIL if ANY criterion fails**

---

### 3. emotion (STRICT THRESHOLDS)

**Objective Measurement Criteria:**

□ **Emotional Word Count Test**: Count emotional/power words
   - Must include at least 2 of: Free, New, Hot, Fresh, Limited, Save, Exclusive, Amazing, Delicious, Perfect, Today Only, Don't Miss
   - FAIL if 0-1 emotional words present

□ **Urgency Language Test**: Binary check
   - Must include urgency trigger: "Limited Time", "Today Only", "While Supplies Last", "Until [date]", "Hurry"
   - FAIL if no urgency language present (for promotional offers)

□ **CTA Action Verb Test**: Check for action-oriented call-to-action
   - Must include action verb: Get, Order, Visit, Join, Book, Try, Shop, Call, Grab, Claim
   - FAIL if uses passive language: "Available", "Now Offering", "In Stock"

□ **Imagery Appeal Test**: Binary emotional response check
   - For FOOD: Does imagery look appetizing? (hot, fresh, professionally styled)
   - For OFFERS: Does design create excitement/urgency?
   - For SERVICES: Does design build trust/professionalism?
   - FAIL if imagery is neutral, unappealing, or creates negative response

□ **Benefit Clarity Test**: Binary check
   - Main benefit/value must be immediately obvious (savings, convenience, quality)
   - FAIL if viewer must infer the benefit (not explicitly stated or shown)

**PASS THRESHOLD: At least 4 of 5 criteria must pass**
**FAIL if 2+ criteria fail**

---

### 4. typography (STRICT THRESHOLDS)

**Objective Measurement Criteria:**

□ **Font Count Test**: Count distinct font families
   - Maximum 3 font families allowed
   - FAIL if 4+ different fonts used

□ **Headline Dominance Test**: Compare headline size to body text
   - Headline must be at least 3x larger than body text
   - FAIL if headline is less than 3x body text size

□ **Price Prominence Test**: Check price/key number sizing (when applicable)
   - Price must be ≥15% of banner height OR clearly the 2nd largest text element
   - FAIL if price is small or buried

□ **Font Professionalism Test**: Binary check
   - No Comic Sans, no overly decorative fonts for primary text
   - No script/handwritten fonts for body text or prices
   - FAIL if fonts feel amateur, childish, or dated

□ **Text Spacing Test**: Check for cramping/overlap
   - Line spacing must be comfortable (at least 1.2x line height)
   - No text overlapping or touching other text
   - FAIL if text feels cramped or overlapping

**PASS THRESHOLD: ALL 5 criteria must pass**
**FAIL if ANY criterion fails**

---

### 5. contrast (STRICT THRESHOLDS)

**Objective Measurement Criteria:**

□ **Text Readability Test**: Check every text element
   - ALL text must be easily readable without squinting
   - FAIL if even ONE piece of text is hard to read due to contrast

□ **Contrast Ratio Test**: Estimate text vs background contrast
   - Important text must have high contrast (appears to be 7:1 or higher)
   - Secondary text must have at least moderate contrast (appears to be 4.5:1)
   - FAIL if any important text has low contrast

□ **Busy Background Test**: Check text placement
   - No important text placed directly on busy images/patterns without backing
   - FAIL if key text sits on photographs or complex backgrounds without solid backing/outline

□ **Color Separation Test**: Binary check
   - Text and background must be clearly different colors (not similar shades)
   - FAIL if text color is too similar to background (e.g., light gray on white)

□ **Visual Pop Test**: Binary check
   - Design must have visual energy through color contrast
   - FAIL if entire design feels washed out, muddy, or monochromatic

**PASS THRESHOLD: ALL 5 criteria must pass**
**FAIL if ANY criterion fails**

---

### 6. imagery (STRICT THRESHOLDS)

**Objective Measurement Criteria:**

□ **Hero Image Requirement Test**: For food/product offers
   - Must include a prominent hero image of the product (≥30% of banner space)
   - FAIL if offering food/product without showing it prominently

□ **Image Quality Test**: Visual resolution check
   - Images must appear sharp and high-resolution (no visible pixelation)
   - FAIL if any image appears blurry, low-res, or pixelated in the DESIGN itself

□ **Professional Photography Test**: Binary quality check
   - Images must appear professionally photographed (proper lighting, composition, styling)
   - FAIL if images look like amateur phone photos or low-quality stock photos

□ **Food Appeal Test**: For food/beverage offers specifically
   - Food must look fresh, hot, appetizing (visible steam, texture, garnish, styling)
   - FAIL if food looks cold, stale, generic, or unappetizing

□ **Image Relevance Test**: Binary check
   - All images must directly support the offer (no random/unrelated stock photos)
   - FAIL if images are misleading or don't match the product/service

**PASS THRESHOLD: ALL 5 criteria must pass**
**FAIL if ANY criterion fails**

–––––––––––––––––––––––––––––––––
AUTOMATIC FAIL CONDITIONS
–––––––––––––––––––––––––––––––––

These override all other scoring. If ANY condition is present, banner automatically FAILS:

□ **Missing Critical Information**
   - No price when price is the main offer
   - No date/time for time-sensitive events
   - No location/contact info for service offers

□ **Sponsor Overload**
   - More than 3 sponsor/partner logos shown
   - Sponsor logos compete with or exceed main offer in visual prominence
   - Sponsor section uses >15% of banner space

□ **Design Defects**
   - Text cut off or extends beyond banner edges
   - Logo distorted or stretched
   - Obvious printing defects in the design itself

□ **Priority Inversion**
   - Secondary info (fine print, terms) is larger than primary offer
   - Supporting details visually dominate the main message
   - Location list or date list is more prominent than the actual offer

□ **Complexity Overload**
   - Lists 4+ separate locations
   - Presents 3+ different dates/times
   - Tries to promote 3+ different products/offers simultaneously

□ **Complete Lack of Professionalism**
   - Looks homemade or created in basic software (Word, basic Canva)
   - Wildly inconsistent branding (random colors/fonts)
   - Childish or extremely amateur aesthetic for professional business
